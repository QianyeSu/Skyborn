模块索引
========

Skyborn 包含以下主要模块：

.. toctree::
   :maxdepth: 2

   ../api/index

核心模块
--------

- **转换模块** (:doc:`../api/conversion`) - GRIB 到 NetCDF 格式转换
- **计算模块** (:doc:`../api/calculations`) - 数据处理和计算功能
- **梯度模块** (:doc:`../api/gradients`) - 梯度计算函数
- **因果分析** (:doc:`../api/causality`) - 时间序列因果关系分析
- **插值模块** (:doc:`../api/interpolation`) - 数据插值和重网格化
- **绘图模块** (:doc:`../api/plotting`) - 数据可视化工具

模块概述
--------

转换模块
~~~~~~~~

提供 GRIB 到 NetCDF 格式的转换功能，支持：

- 单文件转换
- 批量转换
- 自定义参数配置
- 高精度输出

计算模块
~~~~~~~~

包含常用的数据处理函数：

- 经度范围转换
- 线性回归分析
- 数据预处理

梯度模块
~~~~~~~~

计算不同方向的梯度：

- 经向梯度
- 纬向梯度
- 垂直梯度
- 通用梯度计算

因果分析模块
~~~~~~~~~~~~

时间序列因果关系分析：

- Liang 信息流分析
- Granger 因果检验
- 显著性测试

插值模块
~~~~~~~~

数据插值和重网格化：

- 混合坐标插值
- 多维数据插值
- 网格重映射

绘图模块
~~~~~~~~

专业的气象数据可视化：

- 曲线箭头图
- 自定义绘图工具
- 科学绘图增强
